Get started

Choose your installation method.

Claude Code Plugin Recommended

Primary method for Claude Code users. Installs everything automatically.

/plugin marketplace add b17z/sage
/plugin install sage

Gets you: MCP server + hooks (auto-checkpointing)

PyPI

For standalone CLI usage outside Claude Code.

$ pip install claude-sage

Gets you: sage CLI commands (sage list, sage ask <skill> "query", etc.)

Want MCP/hooks in Claude Code? Also run: sage mcp install && sage hooks install

Build from Source

For contributors or custom setups.

$ git clone https://github.com/b17z/sage
$ cd sage && pip install -e ".[dev]"

Then optionally: sage mcp install && sage hooks install
